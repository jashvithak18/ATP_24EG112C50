The application is used to manage employee details such as adding employees, updating information, viewing records, and handling employee-related operations.
---
## Technologies Used

### Frontend
- React.js
- Vite
- CSS

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

---

## Project Structure

### Backend Files

- `APIs/`  
  Contains API route files and request handling logic.

- `middlewares/`  
  Middleware functions for request processing and authentication.

- `models/`  
  MongoDB database models and schemas.

- `.env`  
  Environment variables configuration.

- `server.js`  
  Main backend server file.

- `req.http`  
  API testing requests.

---

### Frontend Files

- `public/`  
  Static assets and public files.

- `src/`  
  Main React source code including components and pages.

- `index.html`  
  Main HTML file.

- `vite.config.js`  
  Vite configuration setup.

---

## Features

- Employee record management
- Add and update employee details
- REST API integration
- MongoDB database connection
- Frontend and backend connectivity
- Full-stack MERN application structure
