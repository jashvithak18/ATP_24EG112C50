
 The project includes HTML pages and CSS styling files used to understand webpage structure, layouts, and styling techniques.
## Files Included

- `index.html`  
  Main homepage of the project.

- `index2.html`  
  Additional HTML practice page.

- `index3.html`  
  Webpage created for practicing HTML concepts.

- `index4.html`  
  Another practice webpage with different elements and layouts.

- `indexhw.html`  
  Homework/practice HTML page.

- `style.css`  
  Main CSS file used for styling webpages.

- `stylehw.css`  
  Additional CSS styling file for practice and homework pages
## Technologies Used
- HTML5
- CSS3
