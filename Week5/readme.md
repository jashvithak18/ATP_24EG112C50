
## About
Different table layouts and designs were created to practice frontend development concepts using HTML, CSS, and Tailwind CSS.
The project includes table examples with separate styling files for learning webpage structure and design.
---

## Files Included
- `Tables.html`  
  Basic HTML table design.

- `Tables.css`  
  Styling file for `Tables.html`.

- `Tables2.html`  
  Second table layout practice page.

- `Tables2.css`  
  Styling file for `Tables2.html`.

- `Tables3.html`  
  Advanced table design practice page.

- `Tables3.css`  
  Styling file for `Tables3.html`.

- `index.html`  
  Main entry page of the project.

- `tailwind.config.js`  
  Tailwind CSS configuration file.

- `vite.config.js`  
  Vite configuration setup.

- `package.json`  
  Project dependencies and scripts.


## Technologies Used
- HTML5
- CSS3
- Tailwind CSS
- Vite
